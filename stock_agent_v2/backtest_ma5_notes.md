## 2. 원래 가설 (사용자 직관)

> "일봉에서 5일선을 계속 따라가면 상승모멘텀이 있고, 5일선으로부터 이격이 심하면 약간의 조정이 온다."

이를 두 명제로 분리:
- **A (추종)**: 종가가 항상 5일 평균선 이상인 경우만 → 다음 5/10일 수익률 ↑
- **B (이격)**: 가격이 MA5 에서 ATR 단위로 멀어질수록 → 다음 5/10일 수익률 ↓ (조정)

사용자 결정 (Round 1):
- 이격 임계값: **ATR 기반**
- 추종 조건: **종가가 항상 MA5 이상인 연속일수 (streak)**
- Forward 기간: **5일, 10일**
- 구현 형태: **단일 ad-hoc 스크립트** (`backtest_ma5.py`)

## 4. Round 2 — A + B 조합 (다음 작업)

### A. 단기 조정 (1~3일) 검증
**가설**: 사용자 원래 직관(이격 → 조정)이 사실은 **다음날~3일 timeframe** 에서만 통할 수 있음. 4+ ATR 이격된 다음날엔 작은 음봉, 그 후 다시 상승.

**측정**:
- forward 1d / 2d / 3d 수익률 추가
- 가설 A 의 streak 버킷 + 가설 B 의 gap_atr 버킷 둘 다에 적용
- 5d / 10d 와 비교해서 "단기 조정이 나타나는 timeframe 이 있는지"

**기대 결과 패턴 예시**:
- 4+ ATR 의 1d median: 음수 또는 0 근처 (단기 조정)
- 4+ ATR 의 3d median: baseline 수준
- 4+ ATR 의 5d/10d median: 큰 양수 (이미 검증됨)
- 이러면 "이격 후 단기 매도, 며칠 뒤 재진입" 같은 룰 도출 가능

### B. Drawdown / 테일 리스크
**가설**: median +4% 라도 max drawdown 은 -10% 일 수 있음. 익절/손절 룰 설계용.

**측정**:
- 각 이벤트(streak/gap 조건 충족일) 다음 10일 동안의 **종가 기준 최대 낙폭(MDD)**:
  - `mdd_10d = min(close[t+1..t+10]) / close[t] - 1`
- 각 이벤트의 다음 10일 **최대 상승폭**:
  - `mfe_10d = max(close[t+1..t+10]) / close[t] - 1`
- 버킷별 mdd / mfe 분포 (median, p25, p75)

**기대 결과 패턴 예시**:
- 4+ ATR 의 10d median return +4% 인데 mdd p25 가 -8% 라면, 평균은 좋지만 변동성 매우 큰 베팅
- streak 21+ 일은 mdd 가 작고 mfe 가 큰 비대칭이라면, 안전한 모멘텀 진입

### 구현 계획
1. `backtest_ma5.py` 수정 (또는 `backtest_ma5_v2.py` 신규):
   - `add_forward_returns()` 에 1d, 2d, 3d 추가
   - `add_forward_extremes()` 신규 — `min_close[t+1..t+N]`, `max_close[t+1..t+N]` 계산
   - `_stats()` 에 mdd_med, mdd_p25, mfe_med, mfe_p75 추가
2. 출력 테이블 확장: 각 버킷에 1d/2d/3d/5d/10d median + 10d mdd_p25 + 10d mfe_p75
3. 차트: 단기 조정 패턴 (1d~10d 시계열 medians 라인 차트), 그리고 mdd vs mfe scatter
4. 결과 PNG: `backtest_ma5_v2_result.png`

### 보완 검토 사항 (선택)
- **시장 분리 (KR vs US)**: universe 편향 완화 위해 결과를 KRX/NASDAQ-NYSE 별로 분리. config.is_overseas() 로 분리 가능.
- **시기별 분리**: 2024 vs 2025+ — 시장 국면 다를 수 있음.
- **표본 부족 버킷 통합**: 3~4 ATR + 4+ ATR → 3+ ATR 단일 버킷 (n=207).
